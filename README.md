# linkpilot-v1
LinkPilot is a lightweight WordPress plugin that uses the DeepSeek API to generate smart internal link suggestions for posts and pages. Built for personal SEO automation and experimentation, showcasing clean PHP development and AI-powered content analysis.

# LinkPilot v2

AI-powered internal linking for WordPress.
Analyzes posts + products, builds topic clusters, and inserts smart internal links automatically.

Features:

Bulk interlinking

Product → blog linking

Auto-styled links (arrow + new tab)

Link log

Undo (single + bulk)

How it works:
Analyze → Suggest → Insert → Undo anytime.

Tech:
Powered by DeepSeek (via OpenRouter).
Inserted links use linkpilot-auto-link.

Install:
Upload ZIP → Activate → Open LinkPilot.
